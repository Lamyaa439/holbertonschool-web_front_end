# JavaScript - DOM Manipulation

This project contains JavaScript exercises focusing on DOM manipulation techniques.

## Learning Objectives
- How to select HTML elements in JavaScript
- How to manipulate DOM elements
- How to fetch data from APIs
- How to update HTML content dynamically

## Requirements
- Chrome browser (version 57.0 or later)
- Semistandard compliant code
- No use of `var` keyword

## Files
- `0-script.js`: Updates header text color to red
